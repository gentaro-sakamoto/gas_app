### Requirements
- node.js
- npm
- clasp

### Setup
```
cd dog_runs/
npm install
```

### Usage
#### Authorize clasp
```
clasp login
```

#### Build your code
```
npm run build
```

#### Push your code to GAS
```
npm run push
```

#### Pull your code from GAS
```
npm run pull

```
